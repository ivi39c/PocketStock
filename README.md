A mobile-friendly inventory management system powered by GitHub Pages, Google Sheets, and LINE Notify.
（一個結合 GitHub Pages、Google 試算表與 LINE 通知的行動友善庫存管理系統。）

PocketStock：放進口袋的即時庫存小幫手！
專為手機操作設計的輕量庫存管理工具。不用下載 App，打開網頁就能隨手更新庫存、掃描編號，還能依據有效期限自動排序。只要庫存有變動或即期，系統就會自動發送 LINE 通知，讓你用一張 Google 試算表，就能完美控管所有物料！

💡核心亮點摘要
手機優先 (Mobile-First)：大按鈕與直覺日曆設計，單手就能輕鬆流暢操作。
零成本後端 (Serverless)：利用免費的 GitHub Pages 搭配 Google Apps Script，免伺服器維護成本。
智慧期限排序：過期或即期品自動置頂，剩餘一個月自動亮紅字防呆。
即時 LINE 連動：新增、修改、刪除庫存時，群組即時收到推播通知。
